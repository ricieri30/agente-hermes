# 🤖 Hermes Agent - Automação com IA Local

Agente de IA projetado para realizar tarefas complexas via Terminal e Browser.

## 🚀 Como Iniciar

1. Clone o repositório:
   ```bash
   git clone https://github.com/ricieri30/agente-hermes.git
   cd agente-hermes
   ```

2. Configure o ambiente:
   ```bash
   cp .env.example .env
   # Edite o .env com suas chaves de API
   ```

3. Inicie com Docker:
   ```bash
   docker-compose up -d
   ```

4. Acesse o Dashboard:
   Abra `http://localhost:3000` no seu navegador.

## 🛠️ Tecnologias
- **Engine:** Hermes 3 (Llama 3.1)
- **Interface:** Dashboard Web (Node.js/Express)
- **Tools:** Playwright (Browser), Bash (Terminal)
- **Runtime:** Docker / Docker Compose
